How to Clone
1. 미리 3D프로젝트를 하나 생성한다.
2. 생성한 프로젝트를 파일탐색기에서 열어 해당 프로젝트 폴더에 들어간다.
3. 해당 프로젝트 폴더에 Git Clone한다.
4. Clone한 폴더 안에 있는 Asset폴더들을 생성한 프로젝트의 Assets폴더안으로 옮긴다. (.git파일도 함께)

How to Build
1. 프로젝트를 들어가 File - Build And Run 메뉴를 선택해 빌드한다
2. 빌드한 프로그램을 실행시켜 플레이한다.

How to Play

1. Play 버튼 클릭시 캐릭터 선택화면으로 넘어감
2. 캐릭터 선택화면에서 캐릭터 선택시 게임 화면으로 넘어감
3. 왼쪽 중단에 Start버튼을 누르면 게임 시작
4. 웨이브마다 몰려오는 적들 처치하면 승리!
5. 중앙의 머리카락 또는 플레이어의 체력이 0이되면 Game Over
6. 매 웨이브가 끝날 때마다 아이템을 통해 강화한다.
